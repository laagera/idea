Inspection styles
=================
To define the custom inspection styles you need to modify the editor.xml file located in your IntelliJ configuration
folder on sub path ./config/options

Windows: C:\Users\#user#\.IntelliJIdea2019.3\config\options
OSX: /Users/#user#/Library/Preferences/IntelliJIdea2019.3/options
Linux: ?

You can either overwrite the existing xml file or simply overwrite the whole xml tag <component name="InspectionProfileManager">
